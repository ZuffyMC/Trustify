# Security Policy

## Reporting Security Issues

If you discover a security issue related to Trustify, please report it privately instead of posting it publicly.

Include:

- Description of the issue
- Steps to reproduce
- Server version
- Trustify version
- Any relevant logs

## Supported Versions

| Version | Supported |
| --- | --- |
| v1.0 Beta | Yes |

## Notes

Trustify should not be used to store sensitive personal information. The plugin is intended for Minecraft server reputation, trust, rating, and community moderation data.
