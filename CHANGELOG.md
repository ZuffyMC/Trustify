# Changelog

## Trustify v1.0 Beta

Initial beta release of Trustify.

### Added

- Reputation system
- Trust system
- Karma system
- Player rating system
- Trust status detection
- Newbie-friendly reputation tracking
- Helpful player detection concept
- Leaderboard system
- Reward system
- Admin GUI for statistics, logs, performance, modules, and player management
- Clean GUI layout with dashboard, profile, rating, rewards, help center, and trust guide
- English and Indonesian message support
- Configurable `config.yml`, `layout.yml`, `rewards.yml`, `status.yml`, and module files
- SQLite database support inside the `data/` folder
- Optional MySQL configuration
- PlaceholderAPI support
- Performance-friendly cache and async database handling

### Notes

This is a beta release, so some features may still need testing, balancing, and improvements.
