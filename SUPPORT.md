# Support

Need help with Trustify?

## Before Asking

Please prepare:

- Server version
- Java version
- Trustify version
- Error log if available
- Steps to reproduce the issue
- List of installed plugins if the issue may be related to compatibility

## Bug Reports

```text
Trustify version:
Server software:
Minecraft version:
Java version:
Error log:
What happened:
Expected behavior:
Steps to reproduce:
```
