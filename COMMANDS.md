# Commands

## Player Commands

| Command | Description |
| --- | --- |
| `/trustify` | Open the main Trustify dashboard |
| `/trustify profile` | View your own Trustify profile |
| `/trustify <player>` | View another player's profile |
| `/trustify rating` | Open the online player rating list |
| `/trustify rating <player>` | Rate a specific player |
| `/trust <player>` | Check a player's trust status |
| `/karma <player>` | Check a player's karma |
| `/rate <player>` | Open the rating GUI for a player |
| `/top trustify` | View top reputation leaderboard |
| `/top trusted` | View top trust leaderboard |
| `/top karma` | View top karma leaderboard |

## Admin Commands

| Command | Description |
| --- | --- |
| `/trustifyadmin` | Open admin dashboard |
| `/trustifyadmin gui` | Open admin dashboard |
| `/trustifyadmin reload` | Reload configuration and messages |
| `/trustifyadmin set <player> reputation <amount>` | Set reputation score |
| `/trustifyadmin set <player> trust <amount>` | Set trust score |
| `/trustifyadmin set <player> karma <amount>` | Set karma score |
| `/trustifyadmin add <player> <type> <amount>` | Add score to a player |
| `/trustifyadmin reset <player>` | Reset player data |
| `/trustifyadmin logs <player>` | View player logs |
| `/trustifyadmin debug <player>` | Debug a player profile |
| `/trustifyadmin performance` | View performance status |
